## How to Take a Good Screenshot

1. **Show the filter bar** — so the viewer can see exactly what filter was applied
2. **Show the full packet list** — at least 3–5 rows visible with the relevant packet selected
3. **Show the packet detail pane** — expanded to the relevant section
4. **Annotate with arrows or boxes** — highlight the specific field being demonstrated

## Tools for Annotation

- **Windows:** Snipping Tool (Win + Shift + S), then open in Paint
- **macOS:** Screenshot (Cmd + Shift + 4), then open in Preview to annotate
- **Linux:** Flameshot (`sudo apt install flameshot`)
